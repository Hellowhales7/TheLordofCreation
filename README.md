# Game Overview 
뱀서류의 로그라이크 게임 만물의 영장입니다. 스팀을 통한 멀티플레이를 지원합니다. 

### 줄거리
인간이 먹이사슬의 최하층이 된 세상 다시 다른 생명체들을 정복해가며 인간의 위상을 높여야 합니다. 당신을 먹잇감으로 생각하는 녀석들을 혼내주고 누가 포식자인지 가르쳐주세요.
